# fun_with_traingle
<hr>

It is an application where four applications are combined which are on triangle.
1. Is Traingle? 
   Here We take three inputs for angles of a triangle and then gives output if it is a triangle or not
2. Quiz on traingle
   Some questions on triangle are asked and if you give answers it will show up the score 
3. Hypotenuse of a triangle
   Lengths of two sides of a triangle taken as input and it's hypotenuse length is given as output
4. Area of a triangle
   Base and height is taken as input and it's area is output shown
